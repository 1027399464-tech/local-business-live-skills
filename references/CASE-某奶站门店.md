# 案例：某奶站门店 · 直播获客方案

> 商家：某奶站门店（酸奶/乳品门店）
> 诊断人：李鑫
> 时间：2026年6月
> 方案状态：已交付，待跟踪数据

---

## 诊断过程

1. 进店吃饭+深度聊天，了解老板的生意：供应链模式、日客流50-60人、客单价~80元
2. 看直播间：没贴片、没固定开播时间、没主推品、话术随意 → 判断"瞎播"
3. 关键发现：核销率高，到店后都或多或少买点 → 店没问题，缺的是拉人
4. 定方向：直播间只负责获客，不负责转化

## 方案

**引流品：** 9.9四个酸奶（前端亏，后端靠到店消费赚回来）

**直播节奏：** 3轮福袋×放单循环，30-40分钟

| 轮次 | 时间 | 动作 |
|------|------|------|
| 第1轮 | 0-10min | 发福袋(10个鸡蛋)→等人进→塑品→放9.9酸奶单 |
| 第2轮 | 10-20min | 开福袋→发新福袋→逼单→推升单换购 |
| 第3轮 | 20-30min | 开福袋→发新福袋→最后冲刺→收尾 |

## 逐字稿

```
开场钩子：
"XX的邻居们，我是楼下奶站老张。"
"今天头一回直播，不整虚的，上来先送福利。"

发福袋：
"左上角福袋我已经发出去了，10个鸡蛋，免费抽。"
"你只要点关注亮灯牌就能参与，倒计时5分钟。"

等福袋期间做停留：
"福袋在抽着，大家都别走。万一中奖的就是你呢？"
"中了你连单都不用下了，白拿10个鸡蛋回家。"

塑品：
"平时你去超市买四个酸奶多少钱？二十多打不住吧？"
"今天我补贴——四个酸奶，九块九。"

放单：
"好，现在开始放今天的同城福利——四个酸奶九块九，就30份。"
"仅限同城，拍了去店里拿。链接在下面，点进去直接拍。"

到店引导：
"拍了的朋友记住——店在XX路XX号，到店说直播间拍的，核销码给收银扫一下。"

收尾：
"好，今天直播就到这儿。拍了酸奶的来店里拿；中奖的来领鸡蛋。"
"没抢到的点关注，下周同一时间还有福利。"
```

## 开播前操作清单

- [ ] 手机架好，冷柜前/店门口，背景露出店招
- [ ] 9.9酸奶链接已上架
- [ ] 福袋已设置（10个鸡蛋，5分钟倒计时）
- [ ] 私域群已发预告："4点开播，有福利"
- [ ] 开播前2小时发一条预告短视频

## 直播中红线

- ❌ 不要开场就卖货——先发福袋等人进，等人齐了再放单
- ❌ 福袋没开之前不要放单——福袋是留人工具，放了单人就走了
- ❌ 每轮放单后立刻预告下一轮福袋——让人有理由继续等
- ❌ 不要读数据，不要看在线人数——专注说话，数据下播再看

## 关键数据目标

| 指标 | 目标 |
|------|------|
| 引流品核销率 | ≥ 60% |
| 到店升单率 | ≥ 30%（老板自己干） |
| 单场新增关注 | ≥ 50个 |
| 平均停留时长 | ≥ 60秒 |
