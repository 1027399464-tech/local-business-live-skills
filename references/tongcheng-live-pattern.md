# 同城直播福袋循环话术模板

> 适用于本地生活商家同城直播（奶站/餐饮/美业等）
> 核心逻辑：福袋留人 → 等人进齐 → 放单转化 → 福袋续流 → 循环

## 节奏框架（30分钟=3轮）

| 时间段 | 做什么 | 目的 |
|--------|--------|------|
| **前5分钟** | 发福袋（免费鸡蛋/实物） | 留人——倒计时让观众舍不得走，积累停留数据 |
| **后5分钟** | 放单（9.9引流品） + 逼单 + 升单钩子 | 转化——等人齐了再卖，不急着出货 |

**关键话术策略：**
- "福袋在抽着，说不定中奖的就是你，中了连单都不用下了"
- 这句话让观众有理由留在直播间等开奖，停留数据自然上去
- 福袋没开之前不卖货——福袋是留人工具，放了单人就走了

## 3轮循环

```
第1轮  0-10min  发福袋→等人进→塑品（为什么便宜）→放9.9引流品
第2轮 10-20min  开福袋→新福袋→逼单→推到店换购升单
第3轮 20-30min  开福袋→新福袋→最后冲刺→收尾
```

## 逐段话术要点

### 开场钩子（30秒）
> 地理锚点 + 福利钩子 + 安全承诺

"XX的邻居们，我是楼下XX店老板。今天头一回直播，先送福利——左上角福袋发了X个XX，点关注亮灯牌就能抢。"

### 等福袋期间（2-3分钟）
> 做停留数据+互动

"福袋在抽着，说不定中奖的就是你。中了你连单都不用下了。XX路住的？打在公屏上我看看。"

### 福袋开奖后放单
> 福袋开了再放单，此时在线是峰值

"好！福袋开了！恭喜中奖的。没中的别走，今天还有XX福利——平时卖XX，今天直播间只要XX。"

### 到店核销引导
> 核销率是核心KPI，不是GMV

"拍了的朋友记住：到店说直播间拍的，核销码给收银扫一下就行。XX路XX号，今天下单今天拿。"

### 升单钩子（到店换购）
> 到店核销时推，线上不卖

"到店核销的时候，再加X块X换一包XX，原价XX，到店才有的福利。"

### 福袋续流
> 放单后立刻预告下一轮福袋，让人有理由继续等

"没抢到的别急，第二轮福袋马上发。先把关注点上，灯牌亮一下。"

## 核心规则

| 规则 | 原因 |
|------|------|
| 不要开场就卖货 | 没人认识你，卖了也白卖，先送福利等人进 |
| 福袋没开之前不放单 | 福袋是留人工具，放了单人就散了 |
| 每轮放单后预告下一轮福袋 | 让人有理由继续停留 |
| 升单话术必须在直播间里说 | 到店再说就晚了，提前种下心理锚点 |
| 不开播前不读数据 | 专注说话，数据下播再看 |

## 适用条件

- **必须**：同城属性（地理锚点有效）、到店核销模式
- **最好有**：私域群可预热（开播前发预告）、近效期/低成本商品做福袋奖品
- **不适合**：纯线上发货（无到店场景）、全国性品类（地理锚点无效）

## 数据目标参考

| 指标 | 参考值 |
|------|--------|
| 引流品核销率 | ≥ 60% |
| 到店换购升单率 | ≥ 30% |
| 新增关注/场 | ≥ 50个 |
| 平均停留时长 | ≥ 60秒 |
