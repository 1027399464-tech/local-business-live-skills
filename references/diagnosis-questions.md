# 标准化进店调研问题清单

> 基于李鑫实战诊断流程结构化，分为6个模块，覆盖从进店到出方案。
> 使用方式：进店后按模块顺序聊，不跳不赶，聊完一圈自然过渡到出方案。

---

## 模块 A：破冰与基本面（5-10分钟）

**目标：** 让老板放松，不像是被审问，同时获取基本信息。

**开场：**
> "老板，你这店开了几年了？看着位置挺不错的。"

**递进：**

| # | 问题 | 你要听什么 |
|---|------|-----------|
| A1 | 你这店开多久了？就这一个店还是有别的？ | 判断"开店1-3年"还是老店 |
| A2 | 一天大概多少客人进店？ | 日均进店人数 → 经营热度 |
| A3 | 客单价大概多少？一个人进来消费多少钱？ | 客单价 → 判断利润空间 |
| A4 | 回头客多不多？大概什么比例？ | 复购率 → 判断店的产品力 |
| A5 | 跟刚开那会比，生意是好了还是差了？ | 趋势 → 焦虑程度 |

**关键判断：**
- 如果老板说"还行"、"差不多" → 他可能不觉得有问题，你需要帮他发现问题
- 如果老板主动吐苦水 → 痛点已经在了，听他说的方向就是你的切入点

---

## 模块 B：线上意识与尝试（5-8分钟）

**目标：** 判断老板对线上渠道的态度和已有基础。

| # | 问题 | 你要听什么 |
|---|------|-----------|
| B1 | 以前试过抖音/直播/短视频吗？ | 0经验 vs 试过但没结果 |
| B2 | （试过）当时怎么做的？自己拍还是找人？ | 是认真在做还是随便玩玩 |
| B3 | （没试过）是不想做，还是不知道怎么做？ | 态度：抗拒还是迷茫 |
| B4 | 你觉得抖音对你这行有用吗？ | 认知层次：信还是不信 |
| B5 | 如果每天花1小时搞线上，你愿意吗？ | 投入意愿 → 判断执行力 |

**关键判断：**
- 试过但放弃 → 他可能踩过坑，知道线上不容易，但更需要外援
- 不知道怎么做 → 你在他的认知空白区，权威感容易建立
- 直接说没用 → 最难转化的类型，优先跳过
- 愿意花时间 → 执行力信号，可以推高客单价方案

---

## 模块 C：产品与引流品（5分钟）

**目标：** 找到可以用来引流的品和升单路径。

| # | 问题 | 你要听什么 |
|---|------|-----------|
| C1 | 什么产品卖得最好？ | 爆品/招牌 → 适合做直播主推 |
| C2 | 有没有那种成本低、但客人特别喜欢的东西？ | 引流品候选（成本可控、有吸引力） |
| C3 | 你现在的团购/套餐是怎样的？ | 如果已经上了团购，看结构和价格 |
| C4 | 客人来了，一般除了点单还会加什么？ | 升单路径 → 到店后的增收点 |
| C5 | 你觉得你的东西跟隔壁比有什么不一样？ | 卖点/差异化 → 直播话术素材 |

**关键判断：**
- 有引流品 + 有升单品 → 直播转化模型完整，只需要执行
- 只有引流品但没升单一 → 到店转化有缺口
- 连引流品都难找 → 需要先帮他想清楚产品结构

---

## 模块 D：4 信号诊断（5分钟）

**目标：** 如果老板已经在播，评估他的直播质量。用观察不用问。

**标准：** 进店后扫一眼直播间画面（如果他在播），或者问他要直播回放。

| 信号 | 好的状态 | 瞎播的状态 |
|------|---------|-----------|
| 贴片 | 有贴片（店名+福利+地址） | 没有贴片，画面干播 |
| 开播时间 | 固定时间开播 | 想播就播，没规律 |
| 主推品 | 有明确的引流品 | 随口讲，没有主推 |
| 话术 | 有框架，有节奏 | 想到哪说到哪 |

**判断逻辑：** 4项里3项不合格 → 核心问题不是内容，是"没有把直播当正经事做"。

**如果不在播：**
> "你现在开了团购/上架了吗？" 
> 用这个代替信号诊断，观察点变成：
> - 有没有团购链接？
> - 链接的图片和描述是否认真？
> - 有没有持续的短视频内容？

---

## 模块 E：核销与到店体验（3分钟）

**目标：** 判断"店能不能接住人"——这是方案方向的核心分水岭。

| # | 问题 | 你要听什么 |
|---|------|-----------|
| E1 | 之前如果有团购，到店核销率大概多少？ | 核销率 → 到店体验和承接能力 |
| E2 | 客人来了之后，除了买团购的，还会消费别的吗？ | 升单率 → 到店转化能力 |
| E3 | 客人走了之后，有没有加你微信？有没有回头？ | 留存 → 你有没有能力把顾客留下来 |

**关键判断（决定方案方向）：**

```
核销率高 + 到店有消费 → 店没问题，只缺客
                 ↓
方案 = 只做拉人（福袋+引流品+放单循环）
不需要复杂话术和人设

核销率低 / 到店不升单 / 没有留存 → 店有问题
                 ↓
方案 = 先解决店的问题（产品调整+服务流程+到店体验优化）
再考虑拉人
```

---

## 模块 F：收尾与报价信号（2分钟）

**目标：** 在聊的自然氛围中，给老板一个"接下来怎么办"的信号。

| # | 动作 | 话术模板 |
|---|------|---------|
| F1 | 总结发现 | "老板，我大概看了一下，你这店的问题是______" |
| F2 | 给方向 | "如果是我来做，我会______" |
| F3 | 出报价 | "我出一份详细的执行方案给你，你照着做就行，299。或者我直接上门陪跑你做，699/两次。你看你选哪个？" |

**关键原则：**
- **不要在模块 A 到 E 之间报价** — 还没建立价值认知就报价 = 等着被拒绝
- **不要把每个模块当考试** — 聊着天自然问完，不是拿来打勾的清单
- **如果老板话多，不用赶** — 有些老板需要先信任后才说实话
- **如果老板防备重，先聊店，不聊线上** — 等他放松了再切到 B 模块

---

## 完整聊天的自然顺序

```
进门 → 看一圈 → 坐下来
  ↓
A: 基本面（轻松聊） → 感觉一下老板状态
  ↓
B: 线上意识（试探态度） → 判断难度等级
  ↓
(如果老板想展示手机) → C: 产品（聊团购/菜单）
  ↓
(如果方便) → D: 直播画面/回放（现场诊断）
  ↓
E: 核销和到店体验（找分水岭）
  ↓
F: 收尾+报价（自然过渡）
```

> 核心心法：不是审问，是对话。你不是在"填写调查表"，是在"帮老板梳理问题"。
> 每个问题之后给一句反馈（"哦，那说明你这个不错" / "这个确实是个问题"），让老板感觉你在帮他看问题，不是在收集信息。
