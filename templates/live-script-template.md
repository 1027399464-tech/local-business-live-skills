# 同城福利直播 · 话术模板

> 适用场景：本地生活商家第一次开播、用低价引流品拉客到店
> 核心逻辑：福袋留人 → 塑品 → 放单 → 升单钩子，每10分钟一轮循环
> 参考案例：某奶站门店（9.9四个酸奶）

## 模板骨架

### 前置准备
- [ ] 选引流品：高频刚需、成本可控、到店自提（如：9.9鲜奶/酸奶/鸡蛋）
- [ ] 设主推品：可从单次消费锁定为订阅（周卡/月卡）
- [ ] 设升单钩：到店核销时才推的换购福利（成本3-5元，原价25）
- [ ] 设福袋：实物福袋（10个鸡蛋/苹果等），5分钟倒计时
- [ ] 架设备：手机支架，背景露出店招/冷柜
- [ ] 私域群发预告：开播前2小时发预告短视频

### 每轮循环（10分钟 = 5分钟福袋 + 5分钟放单）

**福袋部分（0-5分钟）：**
- 发出福袋，告知规则（点关注亮灯牌参与）
- 话术钩子："中了你就不用下单了，白拿"
- 等福袋期间互动：问观众"哪里住的"、"打公屏"
- 福袋开奖：恭喜中奖，预告下一轮福袋

**放单部分（5-10分钟）：**
- 塑品：对比正价，说明为什么便宜（"图你上门"）
- 放单：限时限量，仅限同城，到店核销
- 升单钩：到店加X元换购某商品
- 引导关注：预告下一轮福利

### 三要素（每句话必须至少包含一个）
1. **地理锚点**："XX路""XX小区"——激活同城推荐
2. **时效安全感**："过期自动退"——降低决策门槛
3. **到店场景**："下楼走5分钟"——告诉什么时候来

### 红线
- ❌ 开场就放单——先发福袋等人进
- ❌ 福袋没开就放单——福袋是留人工具
- ❌ 只跟连麦者说话——成交的是观众
- ❌ 升单话术到店再说——必须在直播间提前说
- ❌ 读在线人数——专注说话，下播再看数据

## 配置参数速查

| 参数 | 推荐值 |
|------|--------|
| 福袋频率 | 每10分钟一轮 |
| 福袋奖品 | 实物（鸡蛋/苹果等），价值10-20元 |
| 引流品核销率目标 | ≥60% |
| 到店升单率目标 | ≥30% |
| 投流日预算 | 100元（开播后跑通自然流再加） |
| 投流定向 | 店铺周边3-5km + 25-45岁 + 家庭消费兴趣 |
| 投流时段 | 下午4-6点（下班回家路上） |
