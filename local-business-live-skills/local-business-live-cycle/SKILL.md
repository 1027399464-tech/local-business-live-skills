# 本地生活商家直播全流程

> 定位：你不是IP，你是医生坐诊。本地生活商家的核心不是做大号，是开播能获客、播完能复盘。
>
> 核心链路：短视频引流入场 → 开播承接留人 → 连麦/互动筛客 → 诊断出方案 → 下播复盘迭代

---

## 第零章：核心认知

### 本地生活商家的特殊之处

| 区别 | 传统带货 | 本地生活商家 |
|------|---------|------------|
| 目标 | 卖货冲GMV | 获客+信任建立 |
| 流量 | 泛电商人群 | 同城/附近人群 |
| 转化 | 当场下单 | 到店核销/咨询 |
| 关键指标 | GPM | 商品曝光点击+咨询转化 |
| 复利 | 单品复购 | 回头客+转介绍 |

### 2026年抖音算法新规（影响本地生活商家）

```
推荐得分 = 价值观分 × 内容力权重 × 交易力权重 × 合规系数
```

- **内容力第一权重是场景**：实景溯源（门店实拍/工厂/产地）可获30%流量加权
- **推流机制从实时反馈转向7天慢推流**
- 第1天冷启动：不核心考核GMV，核心考核3秒留存、停留(>25秒)、互动率(>2%)、转粉率(>1.5%)
- 第4-5天：考核复访率、铁粉互动、GPM（需达客单价10倍以上）
- 核心数据权重：3秒留存 > 停留时长 > 老粉互动(权重比新粉高6倍) > 评论互动 > UV价值&GPM > 转化率

## 第一章：线上获客（短视频引流）

> 没有短视频就没有新观众。直播间自然流太慢，出圈短视频是唯一增量来源。

### 1.1 532内容配比

| 类型 | 占比 | 目的 | 拍什么（本地生活版） |
|------|------|------|-------------------|
| 出圈类 | 5条 | 让没看过你的人看见你 | 老板反差/行业秘密/同城热点 |
| 干货类 | 3条 | 让看见你的人觉得你专业 | 怎么选店/怎么避坑/成本揭秘 |
| 人设类 | 2条 | 让觉得你专业的人信任你 | 帮过的案例/老板故事 |

### 1.2 发布节奏

```
开播前2小时：发1条出圈类（为直播蓄水）
开播时：     发1条干货类（预告今日主题）
直播中：     每小时发1条切片（直播精彩片段）
```

### 1.3 "说人话"短视频文案5条铁律

| # | 铁律 | 操作方法 |
|---|------|----------|
| 1 | **脑海有画面** | 写完朗读，自己脑海里能形成具体画面吗？说"好吃"没感觉，说"金灿灿的鸡皮、一股烟往上飘"才有感觉 |
| 2 | **三年级听得懂** | 找目标用户里理解力最弱的人，在心里问他：他能听懂吗？ |
| 3 | **有具体细节** | 不能只写"成本高"，要写"少了一块板、工人再跑一趟、客户三天后再请假回来" |
| 4 | **有故事** | 人类大脑靠共同经验理解抽象概念，故事才能激活想象 |
| 5 | **两个圆重叠** | 你说的 ≠ 他听到的。写完问自己：用户真的收到这个信息了吗？ |

### 1.4 出圈类选题模板

| 钩子类型 | 适合本地生活的示例 |
|---------|------------------|
| 名人反差 | "XXX火锅老板说赚钱靠服务，但你知道他的真实成本是多少吗？" |
| 反认知 | "开了几家店反而越赚越少，你信不信？" |
| 借势热点 | "XX品牌一夜关了XX家店，本地商家能学到什么？" |
| 数字冲突 | "一家店月流水30万，到手不到3万，钱去哪了？" |

### 1.5 付费引流补充

- **谁需要用**：自然流稳定后，需要破层级/修正标签
- **理想占比**：控制在30%以内，理想约20%
- **核心认知**：付费是放大流量的工具，成败根基在于直播间人货场和素材质量
- **本地生活注意**：同城定向要精准，避免浪费在非目标区域

## 第二章：开播（医生坐诊模式）

### 2.1 冷启动期策略（新号前3场）

**特征：** 抖音无法识别直播间定位，初始流量80-90%为不精准人群

**策略：**
1. **先练能力，再求精准流量** — 重点是练主播话术/互动/促单/应对突发
2. **七日螺旋递增** — 盯住停留率、转化率、互动率，逐场提升
3. **合理对待泛流量** — 不精准人群的互动数据也能帮助打标签
4. **流量峰值讲干货，流量下跌抛钩子**

### 2.2 开播前30秒三问自检

| 自检项 | 判断标准 | 修正动作 |
|--------|---------|---------|
| 曝光→进入率 | 前3分钟进线<30人→钩子不够强 | 加重钩子：换痛点表达、重申收益 |
| 评论互动 | 前5分钟0条评论→话太空 | 换成问答：什么行业？几家店？最头疼什么？ |
| 留存时长 | 人均停留<20秒→节奏太慢 | 加快干货节奏，删铺垫，直接进正题 |

> 数据差不可怕，可怕的是没有意识到在前5分钟就修正。
> 每场直播都是独立实验，前一场数据不影响这一场的初始流量池。

### 2.3 整场节奏（90-120min）

```
0-8min     开场钩子 + 核心框架简述（只讲一轮）
8-15min    互动预热：抛问题让观众打公屏 + 念回复 + 引导连线
15-75min   连线期 / B计划（从互动活跃的人里点名邀连线）
75-85min   结尾：关注 + 预告 + 兜底
```

### 2.4 开场10分钟话术骨架

1. **钩子 (30秒)**：用具体数字+反认知制造悬念
2. **干货骨架 (5-8min)**：只讲一轮核心框架
3. **过渡+互动 (2min)**：让观众打行业+问题，念回复建立回应感

### 2.5 连线筛客流程（望闻问切+AB分诊）

**连线前筛选** — 让观众打"什么行业+几家店+最头疼什么"

**5秒判断A/B类：**

| 回答特征 | 类型 | 策略 |
|---------|------|------|
| 有实体店+卡在扩张/管人/分钱 | A类·专科 | 深挖到机制层面，给具体方案（10-20min） |
| 没流量/产品好没人知/想招代理 | B类·全科 | 快问快断给方向，不硬转（3-5min） |

**连线四步：望(观察) → 闻(听完不打断) → 问(挖本质) → 切(双重确认)**

**B类特别重要：** B类问题放大到全场往往有共性——花15秒做【切】，把本来"跑偏"的案例变成公共议题。

### 2.6 无人连线B计划

| B计划 | 内容 | 时长 |
|-------|------|------|
| B1·讲案例 | 从案例库挑一个讲 | 5-8min |
| B2·讲知识点 | 深讲一个具体方法 | 5-8min |
| B3·评论区问答 | "我看看公屏——XXX老板问XXX" | 3-5min |

### 2.7 话术黄金法则（本地生活版）

- **3秒靠画面，15秒靠话术**
- **节奏化设计**：种草→开价→收割
- **情绪把控**：轻松开场→认真介绍→紧迫促销→激情逼单
- **钩子后必须加理由**："这个价格只有今天开播才有"
- **从参数到场景**：不说"纯棉的"，说"周末裹在云朵般柔软的被子里的幸福感"

### 2.8 红线

| 别 | 因为 |
|----|------|
| 开场循环讲框架 | 老粉走了新粉没来 |
| 连线完不提炼直接下一段 | 连线白做 |
| 被问住硬编 | 掏框架反问 |
| 一对一全程不面向观众 | **成交的是观众，不是连麦者** |
| 不拍短视频只靠直播 | 自然流太慢，出圈短视频是唯一增量 |

## 第三章：复盘

### 3.1 复盘六部曲

```
数据复盘 → 节奏复盘 → 商品复盘 → 人群复盘 → 短视频复盘 → 流量结构复盘
   ↓          ↓          ↓          ↓          ↓            ↓
定基调     定成败     找爆款     判精准     挖流量       调比例
```

**第一步 数据复盘：** 看转化率、GPM/客单价、老粉成交占比、停留时长。标出不合格指标。

**第二步 节奏复盘：** 开播极速流接住了吗？放单量是否超过在线5%？是否有连续2-3轮成交递增？

**第三步 商品复盘：** 商品曝光点击率15%合格25%+爆款。曝光高点击低→话术/弹窗问题；点击高转化低→价格/信任问题。

**第四步 人群复盘：** 对比看播vs购买人群画像。无主力人群→需微付费修正标签。

**第五步 短视频复盘：** 筛选基础播放>5000、点击率>5%、5秒完播>30%的视频→开播前抖加加热。

**第六步 流量结构复盘：** 对比各渠道流量占比和支付金额占比。微付费自然流50-70%健康。

### 3.2 本地生活商家关键指标速查

| 指标 | 合格线 | 说明 |
|------|--------|------|
| 曝光进入率 | ~20% | 大众类目~20%；小众10-20% |
| 商品曝光率 | 80-90% | 中控弹窗效率 |
| 商品点击率 | 15%+ | 20%有潜力，25%+爆款 |
| 成交转化率 | 8-10% | 低客单20%合格，高客单5%合格 |
| GPM | ≥客单价×10 | 不达标难获自然流 |
| 老粉成交占比 | ＜20% | 达人/高客单可放宽至50% |
| 停留时长 | ≥30秒 | 单品自然流>35秒，多品>50秒 |
| 转粉率 | ≥3% | 拉新能力指标 |

## 第四章：三频协同（进阶）

本地生活商家流量结构健康参考：

```
自然流（直播推荐）：40-50%
短视频流量：       20-30%
付费流量：         20-30%
```

**三频协同检查清单：**
- [ ] 短视频和直播间货品是否对应？
- [ ] 开播前2小时是否有蓄水视频？
- [ ] 直播中是否有切片发布？
- [ ] 付费占比是否在30%以内？
- [ ] 短视频增长是否带动了自然流增长？
- [ ] 停止付费后自然流是否还稳？

## 附录：常用诊断框架

### 被问住了掏框架

| 框架 | 一句话 |
|------|--------|
| 聚焦三问 | "你赚什么钱？你有什么优势？你缺什么？" |
| 资产就近 | "离你已有的资产越近，成功率越高" |
| 短板外包 | "拿你的长板换别人的长板，别自己补短板" |
| 100选3 | "100个想法筛掉80个，剩20个再筛剩3个" |
| 表面→本质 | "你以为问题是XXX，本质是XXX" |

### 提炼共性模板（连线后30秒必须做）

- **问题归类法**："刚才XXX老板的问题，本质上是XX。各位，多少人也是一样的？"
- **表面→本质法**："很多人以为问题是表面，其实本质是深层。记住一句话：XX"
- **案例→原则法**："这个案例说明什么？不管你做XX还是XX，记住一个原则：XX"

**不提炼 = 连线白做**
