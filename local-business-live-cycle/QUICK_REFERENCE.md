# 本地生活商家直播 · 速查卡

> 打印出来贴墙上，开播前扫一眼

---

## 一、开播前1小时

- [ ] 发1条出圈短视频（为直播蓄水）
- [ ] 准备5-10条素材
- [ ] 确认直播预告已发

## 二、开播前5分钟自检

| 看什么 | 正常吗？ | 不正常怎么调 |
|--------|---------|------------|
| 曝光进入率 | 前3分钟>30人 | 换钩子、换痛点表达 |
| 评论互动 | 有人说话 | 问问题："什么行业？几家店？" |
| 人均停留 | >20秒 | 删铺垫，直接进正题 |

## 三、整场节奏（90分钟）

```
0-8min     开场钩子 + 讲核心框架
8-15min    互动预热：让观众打行业+问题
15-75min   连线 / 案例 / 问答（循环）
75-85min   预告 + 兜底
```

## 四、话术公式

**开场钩子：** 具体数字 + 反认知
> "一家店月流水30万，到手不到3万"

**种草话术：** 不说参数说画面
> ❌ "这是纯棉的"
> ✅ "周末裹在云朵里"

**促单话术：** 钩子 + 理由
> "这个价只有今天开播才有"

## 五、连线筛客（5秒判断）

| 对方说 | 类型 | 怎么接 |
|--------|------|--------|
| 有店 + 卡在分钱/管人 | A类专家 | 深挖给方案（10-20min） |
| 没人知道/产品好卖不出 | B类全科 | 快问快断（3-5min） |

## 六、红线

- ✗ 开场反复讲框架 → 新粉懒得听
- ✗ 连线完不总结 → 白连了
- ✗ 只跟连麦者说话 → **观众才是买单的人**
- ✗ 不拍短视频只靠直播 → 等着喝西北风

## 七、数据合格线

| 指标 | 合格线 |
|------|--------|
| 曝光进入率 | ~20% |
| 商品曝光点击率 | 15%（25%+爆款） |
| 成交转化率 | 8-10% |
| GPM ÷ 客单价 | ≥10倍 |
| 老粉成交占比 | <20% |
| 停留时长 | ≥30秒 |
| 转粉率 | ≥3% |
| 付费占比 | <30% |

## 八、复盘一句话

> 数据差不可怕，可怕的是播完不改。
> 每场都是独立实验，前一场不影响下一场。
