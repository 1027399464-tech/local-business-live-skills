# 本地生活商家直播 · 速查卡

> 打印出来贴墙上，开播前扫一眼

---

## 一、开播前1小时

- [ ] 发1条预告短视频（为直播蓄水）
- [ ] 确认引流品链接已上架
- [ ] 确认福袋已设置（奖品+5分钟倒计时）
- [ ] 私域群发预告

## 二、开播节奏（30-40分钟）

```
第1轮 0-10min  发福袋→等人进→塑品→放引流品单
第2轮 10-20min 开福袋→发新福袋→逼单→推升单换购
第3轮 20-30min 开福袋→发新福袋→冲刺→收尾
```

**核心：** 福袋是用来留人的，人齐了才放单。放单后立刻预告下一轮福袋。

## 三、话术公式

**开场钩子：** 具体数字 + 反认知

**塑品话术：** "平时去超市买多少钱？今天我补贴——XX"

**放单话术：** "同城福利，拍了去店里拿。链接在下面。"

**到店引导：** "到店说直播间拍的，核销码给收银扫一下。"

## 四、开播中红线

- ❌ 不要开场就卖货——先发福袋等人进
- ❌ 福袋没开之前不要放单——放了单人就走了
- ❌ 每轮放单后不预告下一轮——让人没有理由继续等
- ❌ 读数据、看在线人数分心——专注说话，数据下播看

## 五、数据合格线

| 指标 | 合格线 |
|------|--------|
| 曝光进入率 | ~20% |
| 商品曝光点击率 | 15%（25%+爆款） |
| 成交转化率 | 8-10% |
| GPM ÷ 客单价 | ≥10倍 |
| 老粉成交占比 | <20% |
| 停留时长 | ≥30秒 |
| 转粉率 | ≥3% |
| 付费占比 | <30% |

## 六、复盘一句话

> 数据差不可怕，可怕的是播完不改。
> 每场都是独立实验，前一场不影响下一场。
