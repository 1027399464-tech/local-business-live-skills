# 本地生活商家直播全流程

> 定位：你不是帮商家做大号的，你是帮他把直播间变成获客工具。
>
> 核心链路：进店深度聊 → 找4个信号判断问题 → 找关键杠杆 → 设计方案 → 开播执行 → 复盘迭代

---

## 第零章：商家诊断接入流程（实战案例）

> 本章是李鑫第一次给商家做诊断的真实流程。不是你进店该怎么做，而是他实际怎么做。

### 0.1 第一步：先聊透，不是先看号

不是上来打开抖音看数据。先跟老板吃饭聊天，了解：

- 他做什么生意？几个店？在哪儿？
- 一天进店多少人？客单价多少？
- 抖音现在怎么播的？播了多久？
- 他自己觉得最大问题是什么？
- 有什么是他觉得"还行"的地方？

**关键：** 让老板自己说，你不预设框架。他真正焦虑什么、对什么没底，聊出来的信息比任何数据面板都值钱。

### 0.2 第二步：4个信号判断"是不是在瞎播"

不看数据，看画面和操作：

| 信号 | 好的状态 | 瞎播的状态 |
|------|---------|-----------|
| 贴片 | 有贴片（店名+福利+地址） | 没有贴片，画面干播 |
| 开播时间 | 固定时间开播 | 想播就播，没规律 |
| 主推品 | 有明确的引流品 | 随口讲，没有主推 |
| 话术 | 有框架，有节奏 | 想到哪说到哪 |

**判断逻辑：** 4项里3项不合格 → 核心问题不是内容，是"没有把直播当正经事做"。

### 0.3 第三步：找关键杠杆

老板娘说了一句关键信息：**"核销率很高，来了都或多或少买点。"**

这句话决定了整个方案方向：

```
核销率高 + 到店有消费 → 店本身没问题
                 ↓
问题不是"转化"，是"进店人数不够"
                 ↓
直播间只干一件事：拉人进店
                 ↓
不需要复杂话术，不需要人设打造
只需要：福袋留人 + 低价引流品 + 到店核销引导
```

**这是你（李鑫）的判断模型：**
> 先看"店能不能接住人"——如果能，问题就是拉人，不是转化。
> 如果不能，才想话术、人设、信任的问题。

### 0.4 第四步：设计方案（实战版）

**引流品：** 9.9四个酸奶
- 前端不赚钱，目的是让人下单到店
- 成本结构：让老板自己算，你只设计流量逻辑

**直播节奏：** 3轮福袋×放单循环（30-40分钟）

| 轮次 | 动作 | 目的 |
|------|------|------|
| 第1轮 0-10min | 发福袋（10个鸡蛋）→等人进→塑品→放9.9酸奶单 | 蓄水+首单转化 |
| 第2轮 10-20min | 开福袋→发新福袋→逼单→推升单换购 | 续流+制造紧迫 |
| 第3轮 20-30min | 开福袋→发新福袋→最后冲刺→收尾 | 收割尾流 |

**核心逻辑：**
- 福袋是用来留人的工具，人齐了才放单
- 放单后立刻预告下一轮福袋——让人有理由继续等
- 福袋在抽的时候说"说不定中奖的就是你，中了就不用下单了"——这句话让观众有理由留在直播间等开奖

**升单：** 直播间不负责升单，只负责拉人进店。人到了店里，老板自己维护。

### 本地生活商家的特殊之处

| 区别 | 传统带货 | 本地生活商家 |
|------|---------|------------|
| 目标 | 卖货冲GMV | 获客+信任建立 |
| 流量 | 泛电商人群 | 同城/附近人群 |
| 转化 | 当场下单 | 到店核销/咨询 |
| 关键指标 | GPM | 商品曝光点击+咨询转化 |
| 复利 | 单品复购 | 回头客+转介绍 |

### 2026年抖音算法新规（影响本地生活商家）

```
推荐得分 = 价值观分 × 内容力权重 × 交易力权重 × 合规系数
```

- **内容力第一权重是场景**：实景溯源（门店实拍/工厂/产地）可获30%流量加权
- **推流机制从实时反馈转向7天慢推流**
- 第1天冷启动：不核心考核GMV，核心考核3秒留存、停留(>25秒)、互动率(>2%)、转粉率(>1.5%)
- 第4-5天：考核复访率、铁粉互动、GPM（需达客单价10倍以上）
- 核心数据权重：3秒留存 > 停留时长 > 老粉互动(权重比新粉高6倍) > 评论互动 > UV价值&GPM > 转化率

## 第一章：线上获客（短视频引流）

> 没有短视频就没有新观众。直播间自然流太慢，出圈短视频是唯一增量来源。

### 1.1 532内容配比

> 📌 **框架参考**（来源：陈晶IP方法论。李鑫尚未实战验证此配比的有效性）

| 类型 | 占比 | 目的 | 拍什么（本地生活版） |
|------|------|------|-------------------|
| 出圈类 | 5条 | 让没看过你的人看见你 | 老板反差/行业秘密/同城热点 |
| 干货类 | 3条 | 让看见你的人觉得你专业 | 怎么选店/怎么避坑/成本揭秘 |
| 人设类 | 2条 | 让觉得你专业的人信任你 | 帮过的案例/老板故事 |

### 1.2 发布节奏

```
开播前2小时：发1条出圈类（为直播蓄水）
开播时：     发1条干货类（预告今日主题）
直播中：     每小时发1条切片（直播精彩片段）
```

### 1.3 "说人话"短视频文案5条铁律

> 📌 **框架参考**（来源：Vikki内容方法论。李鑫尚未实战验证每一条）

| # | 铁律 | 操作方法 |
|---|------|----------|
| 1 | **脑海有画面** | 写完朗读，自己脑海里能形成具体画面吗？说"好吃"没感觉，说"金灿灿的鸡皮、一股烟往上飘"才有感觉 |
| 2 | **三年级听得懂** | 找目标用户里理解力最弱的人，在心里问他：他能听懂吗？ |
| 3 | **有具体细节** | 不能只写"成本高"，要写"少了一块板、工人再跑一趟、客户三天后再请假回来" |
| 4 | **有故事** | 人类大脑靠共同经验理解抽象概念，故事才能激活想象 |
| 5 | **两个圆重叠** | 你说的 ≠ 他听到的。写完问自己：用户真的收到这个信息了吗？ |

### 1.4 出圈类选题模板

| 钩子类型 | 适合本地生活的示例 |
|---------|------------------|
| 名人反差 | "XXX火锅老板说赚钱靠服务，但你知道他的真实成本是多少吗？" |
| 反认知 | "开了几家店反而越赚越少，你信不信？" |
| 借势热点 | "XX品牌一夜关了XX家店，本地商家能学到什么？" |

### 1.5 付费引流补充

- **谁需要用**：自然流稳定后，需要破层级/修正标签

## 第二章：开播获客（福袋×放单循环）

> 本章基于商家实战方案。适用场景：商家有实体店、有团购品、核销率尚可，但直播间没人进。

### 2.1 整场节奏（30-40分钟）

适合本地生活获客型直播——不长篇大论，每轮10分钟，3轮搞定：

```
第1轮 0-10min  发福袋→等人进→塑品→放引流品单
第2轮 10-20min 开福袋→发新福袋→逼单→推升单换购
第3轮 20-30min 开福袋→发新福袋→最后冲刺→收尾
```

### 2.2 核心逻辑

```
福袋不是奖品，是买停留时长的工具。
        ↓
福袋在抽的时候，人就有理由留在直播间等开奖。
        ↓
人齐了才放单——福袋没开之前不放单。
        ↓
放单后立刻预告下一轮福袋——让人继续等。
```

**关键话术：** "说不定中奖的就是你，中了就不用下单了"——这句让观众有理由留下。

### 2.3 逐轮话术框架

**第1轮（0-10min）：蓄水+首单**

| 时间 | 动作 | 话术要点 |
|------|------|---------|
| 0:00-0:30 | 开场钩子 | "XX的邻居们，我是楼下XX店老板。今天不整虚的，上来先送福利" |
| 0:30-1:00 | 发福袋 | "左上角福袋发了XX福利，点关注亮灯牌就能参与，倒计时5分钟" |
| 1:00-3:00 | 等福袋期间做停留 | "福袋在抽着，说不定中奖的就是你，中了就不用下单了" |
| 3:00-4:00 | 塑品 | "平时去超市买多少钱？今天我补贴——XX价格" |
| 5:00-6:00 | 放单 | "同城福利，拍了去店里拿。链接在下面，点进去直接拍" |
| 6:00-8:00 | 升单钩子 | "到店核销时加X元换购XX——这个是到店才有的福利" |
| 8:00-10:00 | 预告下一轮 | "没拍到的别急，下一轮福袋马上跟上" |

**第2轮（10-20min）：续流+逼单**

- 发新福袋
- 等福袋期间重复塑品+逼单
- 强调到店核销引导
- 福袋开了之后追单

**第3轮（20-30min）：冲刺+收尾**

- 最后一轮福袋
- "还剩最后X份"
- 总结：拍了去哪领、中奖了去哪领
- 预告下次直播时间

### 2.4 开播前准备清单

```
□ 手机架好，背景露出店招
□ 引流品链接已上架
□ 福袋已设置（奖品+5分钟倒计时）
□ 私域群已发预告
□ 开播前2小时发一条预告短视频
```

### 2.5 直播中红线

```
❌ 不要开场就卖货——先发福袋等人进
❌ 福袋没开之前不要放单——放了单人就走了
❌ 每轮放单后不预告下一轮——让人没有理由继续等
❌ 读数据、看在线人数分心——专注说话，数据下播看
```

### 2.6 关键数据目标

| 指标 | 目标 |
|------|------|
| 引流品核销率 | ≥ 60% |
| 到店升单率 | ≥ 30%（老板自己干） |
| 单场新增关注 | ≥ 50个 |
| 平均停留时长 | ≥ 60秒 |

## 第三章：复盘

### 3.1 复盘六部曲

> 📌 **框架参考**（来源：那村儿风起短视频内容。李鑫用过但未完整验证每一步的数字标准）

```
数据复盘 → 节奏复盘 → 商品复盘 → 人群复盘 → 短视频复盘 → 流量结构复盘
   ↓          ↓          ↓          ↓          ↓            ↓
定基调     定成败     找爆款     判精准     挖流量       调比例
```

**第一步 数据复盘：** 看转化率、GPM/客单价、老粉成交占比、停留时长。标出不合格指标。

**第二步 节奏复盘：** 开播极速流接住了吗？放单量是否超过在线5%？是否有连续2-3轮成交递增？

**第三步 商品复盘：** 商品曝光点击率15%合格25%+爆款。曝光高点击低→话术/弹窗问题；点击高转化低→价格/信任问题。

**第四步 人群复盘：** 对比看播vs购买人群画像。无主力人群→需微付费修正标签。

**第五步 短视频复盘：** 筛选基础播放>5000、点击率>5%、5秒完播>30%的视频→开播前抖加加热。

**第六步 流量结构复盘：** 对比各渠道流量占比和支付金额占比。微付费自然流50-70%健康。

### 3.2 本地生活商家关键指标速查

| 指标 | 合格线 | 说明 |
|------|--------|------|
| 曝光进入率 | ~20% | 大众类目~20%；小众10-20% |
| 商品曝光率 | 80-90% | 中控弹窗效率 |
| 商品点击率 | 15%+ | 20%有潜力，25%+爆款 |
| 成交转化率 | 8-10% | 低客单20%合格，高客单5%合格 |
| GPM | ≥客单价×10 | 不达标难获自然流 |
| 老粉成交占比 | ＜20% | 达人/高客单可放宽至50% |
| 停留时长 | ≥30秒 | 单品自然流>35秒，多品>50秒 |
| 转粉率 | ≥3% | 拉新能力指标 |
